# Fake Store Web App 🛍️

A simple **E-Commerce Web Application** built using **HTML, CSS, and JavaScript**.  
This project fetches product data from the **FakeStore API** and displays it in a clean product catalog interface.

The application allows users to **browse products, search items, and view single product details**.
